Recommended Learning Resources

These are some of the best free resources for learning more about web browsers and web development in general. Most of these I've used personally.

MDN Web Docs:Mozilla's documentation is the gold standard reference for HTML, CSS, and JavaScript. If you only bookmark one resource, make it this one.

Visit MDN Web Docs →

web.dev:Google's developer resource. Really strong on performance, PWAs, and modern browser APIs. The articles are well written and practical.

Visit web.dev →

freeCodeCamp:Free, project-based web development curriculum. Great for beginners who want to learn HTML, CSS, and JS by actually building things.

Visit freeCodeCamp →

The Odin Project:A free, open source full stack curriculum. Goes deeper than most beginner resources and doesnt hold your hand too much.

Visit The Odin Project →

W3Schools:Quick reference for HTML/CSS/JS syntax. Not always the most in-depth but great when you just need to look something up fast.

Visit W3Schools →

Inside look at modern web browser (Chrome developers blog):A 4-part series by Mariko Kosaka that goes deep on how Chrome's multi-process architecture works. Highly recommended if this topic interests you.

Read the Series →

Can I Use:A must-bookmark tool. Shows which browser versions support which HTML/CSS/JS features. Essential for cross-browser compatibility work.

Visit Can I Use →

W3C Validator:The official HTML validator. Paste in your code or a URL to check for markup errors. Required for this class, and honestly just good practice.

Visit W3C Validator →
