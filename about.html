About This Project

A bit about me and why I chose web browsers as my research topic.

About Me

My name is Niklas Knipschild. I'm an IT Infrastructure major currently taking IT 3203: Intro to Web Development at KSU.

I actually have some history with web development. When I was younger, I used to build websites on the side for car salespeople I worked with. The idea was simple: a personal website that showcased their inventory, built trust with buyers before they even walked in the door, and drove more leads through good design and SEO. That experience taught me a lot about what makes a website actually work, not just look nice, but convert visitors into customers. Things like page load speed, clear calls to action, mobile-friendliness, and how search engines index content became pretty important to me early on.

That background is why you'll notice this site uses SEO-friendly HTML practices: semantic elements, descriptive page titles, meta descriptions, logical heading structure, and meaningful alt text on images. It's less about this being a web dev class project and more about it being second nature at this point. Feel free to check the source code.

About This Project

This site is my Milestone 1 project for IT 3203. The topic, web browsers as a platform, came from my research paper for the class. The paper explores how the modern web browser evolved from a simple hypertext viewer into the complex, high-performance computing environment that most businesses now run on.

I picked this topic because it connects directly to my IT Infrastructure focus. Understanding how browsers work under the hood. Rendering engines, sandboxing, multi-process architecture, and security models are genuinely useful knowledge for someone working in infrastructure. The browser is now essentially the operating system layer for most enterprise software.

The color scheme and layout structure is carried over from a previous project. I know, purple and gold for a browser research site is a bit of a stretch. But it works, and I wasnt about to redesign the whole thing from scratch.
